# Cron Job
Set Cron Job in your hosting or virtual private server (vps).

* * * * * command to be executed
– – – – –
| | | | |
| | | | +—– day of week (0 – 7) (Sunday=0)
| | | +——- month (1 – 12)
| | +——— day of month (1 – 31)
| +———– hour (0 – 23)
+————- min (0 – 59)

Example : 0 0 * * * /home/AutoSahur/run.php
